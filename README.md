# BiliBall 🎱

Track your billiards games with friends. Create a private league, log results, and see who's really the best.

---

## What is BiliBall?

BiliBall is a web app for groups of friends who play pool together. Instead of keeping score on paper, BiliBall tracks everything automatically — wins, losses, streaks, and head-to-head records.

---

## How it works

### 1. Create an account
Sign up with a username, full name, and password. Your data stays private.

### 2. Create or join a league
One person creates the league and shares an invite code with friends. Everyone joins using that code. The league is private — only members can see the stats.

### 3. Log your games
After each game, the winner logs the result. You can only log games you played in — no one can enter results on your behalf.

### 4. Watch the leaderboard update
The leaderboard updates in real time after every game, ranked by score (wins minus losses).

---

## Features

**Leaderboard**
See all players ranked by score. Each player shows their wins, losses, and overall score. Click on any player to view their full profile.

**Player Profile**
Each player has a profile showing:
- Win rate percentage
- Total wins, losses, and games played
- Score (wins minus losses)
- Win rate against each opponent
- Full game history

**Head to Head**
Pick any two players and see their complete history against each other — total games, wins for each side, and who's currently on top.

**Game Notes**
Add a short note when logging a game ("incredible comeback", "last second win"). Notes appear in game history and can be deleted anytime.

**Delete Games**
Made a mistake? You can delete any game you were part of directly from your profile.

**Dark Mode**
Switch between light and dark theme with one tap.

**Works everywhere**
BiliBall is fully responsive — works on mobile, tablet, and desktop.

---

## Privacy

- Leagues are private. Only members with the invite code can join.
- You can only log games you personally played in.
- Only you can delete your own games and notes.
